Учебный курс по FrontEnd разработке

FrontEnd разработчик создает видимую для пользователя часть веб-страницы.

Для того, чтобы стать FrontEnd разработчиком, не нужны знания математики. Освоить специальность может каждый. Курс подойдёт для мальчиков и девочек старше 14 лет.

Знания, которые ты получешь

  • HTML

    Стандартизированный язык разметки документов для просмотра веб-страниц в браузере. Весь текст, а так же каждый раздел этой веб-страницы это и есть разметка.

  • CSS

    Формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Делает веб-страницу красивой.

  • JavaScript

    Язык программирования. JavaScript обычно используется как встраиваемый язык для программного доступа к объектам и взаимодействия с ними. Во FrontEnd этими объектами являются HTML и CSS.
